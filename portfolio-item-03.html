Data Engineering: Storing Big Data

On-premise Source Connection and Streaming Data Pipeline

This project showcases my proficiency in data engineering through two tasks. In Part 1, I connected on-premise data sources to AWS using CloudFormation and SNS alerts. In Part 2, I implemented a streaming data pipeline to stream synthetic data to an S3 bucket and created custom SNS alerts for pipeline failures.



Technologies Used:

  • AWS CloudFormation
  • Linux
  • YAML
  • Amazon S3
  • AWS Lambda
  • SNS (Simple Notification Service)


Project Outcome:

Successfully connected on-premise sources to the cloud and implemented a robust streaming data pipeline, proving my technical proficiency in data engineering.
