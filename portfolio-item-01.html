JSL Portfolio Piece ProjectAgile Board

Kanban Task Management App

The Agile Board project is a comprehensive Kanban Task Management App developed as part of the JSL course. The primary objective of this project is to simulate a real-world development environment where a junior developer is tasked with bringing a Kanban Task Management App to life. The project involves a blend of debugging, feature implementation, and UI enhancement.



Technologies Used:

  • HTML for structuring the web pages.
  • CSS for styling the application, including responsive design techniques.
  • JavaScript for implementing functionality, handling user interactions, and managing data.
  • Local Storage for persisting data across sessions.


Project Outcome:

By the end of the project, a fully functional Kanban Task Management App was developed. The application allows users to manage tasks efficiently, customize the theme, and ensure data persistence across sessions. This project serves as a testament to my ability to debug, enhance, and develop web applications, preparing me for real-world software development scenarios.
